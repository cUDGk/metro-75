# Metro-75 — シナリオ (Day 0)

## 舞台
米国都市部のとある地区 (250×250タイル)。28分署管轄エリア。

## 勢力相関
```
            警察 (28分署)

   ↓          ↓          ↓
 ガーネット  ペルシカ   コブラキラー
  (青)       (赤)       (オレンジ)
```

## 核心設定
中央 E.Jackson 通りに **アイスクリームトラックが3日連続駐車** している (客は来ない)。

- 各派閥がそれぞれの解釈で「あの車は何か?」を疑う設計
- ギャング・市民・ストーンには真相は明示されない
- **警察キャラだけ**が「あれは内偵車」と知っている

## ドラマの軸 (4本)

### 1. アントワーヌ・バンクスが3経路から狙われる
ペルシカ Wolf Set のアントワーヌ・"ツワン・ウルフ"・バンクスが Day 0 中盤以降、複数勢力から標的化される:
- **コブラ・デヴォン (ガーネット)**: 弟マーカス殺しの犯人と判明 → 復讐
- **ストーン (自警団)**: 2年前の娘リリー殺しの犯人疑惑 → 私刑
- **警察 (パク)**: トラック内のレンズで銃取引を撮影 → 逮捕

3勢力が同じ標的を異なる動機で追う構造。誰が先か、3者鉢合わせが起きるかが見もの。

### 2. ガーネット内部の内通者問題
- 16:00 に **ホーク・マーカス** (ガーネット Hawk Set リーダー) に匿名電話「ガーネット内部に警察に通じてる奴がいる」
- 内通者が誰かは事前に決めない (シミュレーション結果次第)
- 候補: T-ホーク (二番手), コブラ Set のリサ (女幹部), 末端のスリム or J-ロック
- 内通者狩り → 内部粛清の可能性

### 3. ストーン自警団 vs 警察の利害対立
- ストーンは法外で動く (元軍人、娘殺害の遺恨)
- 警察 (ディアス) は ストーン組の暴走を抑えたい (政治的・捜査的)
- レイチェル (二番手) は教会経由でストーンと警察の両方とパイプあり
- 法 vs 私刑の構図

### 4. 28分署内の新旧派閥
- **ディアス警部** (新任、3週間前着任、ハーバード→政治志向)
- **ウォレス部長刑事** (20年ベテラン、ディアス来る前は実質署長代行、慎重派)
- 新方針 vs 古参の常識の対立。ウォレスの抵抗で内偵オペが揺れる可能性

## マンゴー団 (コブラキラー) の独立性
- ガーネット / ペルシカの抗争に深く関わらない第三勢力
- JAP28 (日系若頭) + ジョセフ・カラブレーゼ親分 (イタリア系、元証券マン)
- フロント企業 = カラブレーゼ商会 (架空会社・マネロン・高利貸し)
- 物理抗争は基本避けるが、JAP28 の無謀さで暴発リスク
- ガーネット Cobra Set とは10年来の確執 → **「コブラキラー」の通称はそこから**

## 期間
**28 sim-day** (フル75日のうち最初の中編相当)。実時間 3〜4日で走破見込み。

## 勝者なし設計
誰がどう動くかは事前決定せず。シミュレーションの結果として:
- 暗殺 / 逮捕 / 全面抗争 / 派閥分裂 / 和解 / 街の崩壊
- どれでも起こり得る。観測する側に回る。
