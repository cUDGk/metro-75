# Metro-75 — 要件まとめ (ユーザー仕様)

## ゴール
- 250×250 タイルの米国都市部スライスで、AI エージェント 60人 (アクティブ 17 / NPC 43) が 28 sim-day 自律行動するシミュレーション。
- "Project Sid" (Altera) インスパイアだがコピーではない、独自シナリオ。
- **Local LLM のみ** (OpenAI/Anthropic 等は使わない、検閲回避)。

## モデル / 計算リソース
- **モデル**: `huihui_ai/qwen3-abliterated:30b-a3b` (MoE, 30.5B/3B active, Q4_K_M)
- **ランタイム**: Ollama (Vulkan backend on Win)
- **ホスト 2台ロードバランス**:
  - 主 PC: `http://127.0.0.1:11434` (Vulkan + RX 6600, 16.6 TPS)
  - ミニPC: `http://192.168.1.7:11434` (実測 2.7s/呼び出し、主PC より速い)
  - 環境変数 `OLLAMA_HOSTS=h1,h2` でラウンドロビン + フェイルオーバー

## 言語 / 文体
- **会話 (say / whisper) は完全日本語**
- カタカナ名 + 略称運用 (例: マーカス・"ホーク"・トンプソン → 「ホーク」)
- 派閥名は派生型: ザクロ→**ガーネット**、桃→**ペルシカ**、マンゴー→**コブラキラー**
- 日系キャラは **JAP28 のみ**

## 街の構造
- **大通り** 5タイル幅 × 6本 (3 NS + 3 EW、50タイル間隔)
- **脇道** 2タイル幅 × 多数 (10タイル間隔)
- **大公園 5箇所**: ガリソン (25×25)、イースト・リバー (22×22)、ノースウッド (20×16)、中央広場 (18×14)、リバーサイド芝生 (14×14)
- **公共施設**: 28分署/学校2/図書館/消防署/教会2/病院/銀行/法律事務所
- **商業**: 酒場2/レストラン3/市場3/ガソスタ3/雑貨2/コーナーストア
- **建物バリエーション**: HOUSE_S/M/L + APARTMENT + OFFICE + WAREHOUSE — サイズ・形状ばらばら
- 39ランドマーク + 名なしビル多数

## アーキテクチャ (並列化)
- **3-Phase tick**:
  1. Phase A (sync): 必要 agent 列挙 + 世界スナップショット
  2. Phase B (parallel): ThreadPoolExecutor で全 LLM 呼出を並列 (host pool に律速)
  3. Phase C (sync): 結果を deterministic 順で適用 (位置/メモリ/broadcast の競合排除)
- **LLM ホストプール**: 各ホスト = セマフォ1。同時実行数 = ホスト数。失敗時自動フェイルオーバー
- **SQLite WAL**: per-agent memory の並列読み書き対応

## ロギング (全部保存)
- `events.log` — 全 action / say / witness の時系列ログ
- `dialogue.log` — 会話のみ抽出
- `memory.db` — agent 別の thoughts/reflections/plans (SQLite)
- スナップショット PNG — 4 sim-hour ごと (28日 × 6 = 168枚) もしくは sim-day ごと
- `daily_state_*.json` — sim-day 終了時の全 agent state
- 異常終了対応: tick ごとに save、Ctrl+C で grace shutdown

## 品質要件 (v3 で対応済)
- ループ発言抑止: 類似度 (n-gram overlap) ベースで弾く
- ターン取り: 5タイル以内で話しかけられた agent は次 tick で応答 priority
- アクション語彙検証: 不正な verb は再生成
- 引用符サニタイズ: 「『...』」のネスト除去
- プロンプト圧縮: -30% トークン

## 設計原則
- **勝者なし設計**: 結末を事前に決めない。シミュレーションの結果として暗殺/逮捕/抗争/裏切り/和解いずれもあり得る
- **コンテキスト分離**: 各 agent の記憶は完全独立 (SQLite agent_id カラム)
- **内偵情報の隠蔽**: アイスクリームトラックの正体 (警察の張り込み) は **警察キャラだけ知る**。ギャング/市民は素の観察から推測。これにより各キャラの "振れ" が出る
